Om die een groot EXE file te kry (vs. 'n klein EXE en spul DLL files) moet mens "Qt static linking" doen.
Sien die ander dokumnet vir daai stappe.

Mens kan die finale EXE kleiner maak met UPX (ander programmetjie) maar dan gebruik die EXE meer memory want dit moet basies die hele groot EXE extract in memory. So dis op die ou ent nie regtig baie nice nie...

Ek het Qt 5.6 gebruik (download, install via installer) met Qt Creator vir die IDE.
